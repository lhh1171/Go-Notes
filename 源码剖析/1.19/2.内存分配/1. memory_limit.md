# memory_limit

1.19 版本`GOGC`参数增加了`memory_limit`选项。通过该选项控制Go使用内存总量及调整Go GC垃圾回收器的行为。

该选项有两种形式:

* 通过 `runtime/debug`函数调用 `SetMemoryLimit`。
* 使用 `GOMEMLIMIT` 环境变量。
